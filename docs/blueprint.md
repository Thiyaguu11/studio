# **App Name**: Leafy

## Core Features:

- Chromium Engine: Utilize the Chromium rendering engine for web page display and secure browsing.
- URL Bar: Omnibox at the top for direct URL entry and Google Search.
- Tab Management: Enable multiple tab usage for simultaneous web browsing.
- Navigation Controls: Implement back, forward, and refresh buttons.
- Custom Homepage: Full-screen, dark-themed homepage with centered Google search bar, personalized greeting ("Good Evening, Thiyagu"), and current time display.
- Quick Access Icons: Display icons for Google, YouTube, Gmail, ChatGPT, and GitHub, opening in new tabs.

## Style Guidelines:

- Background color: Dark slate gray (#374151), providing a sleek, dark aesthetic suitable for prolonged screen use.
- Primary color: Soft gray (#D1D5DB), ensuring text and icons are legible against the dark background.
- Accent color: Muted cyan (#67E8F9), used sparingly to highlight interactive elements without causing distraction.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and readable interface. Code font: 'Source Code Pro' for displaying URLs or debugging information.
- Rounded UI elements throughout the browser application.
- Minimal, distraction-free design focused on core functionality.
- Use icons inspired by Apple's Liquid Glass theme, featuring a translucent, layered appearance with subtle gradients and a sense of depth.
